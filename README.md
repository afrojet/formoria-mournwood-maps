# formoria-mournwood-maps
Maps and Handouts for the Mournwood Adventure

---

This repository contains maps and printable handouts for the **Mournwood Adventure**, a shifting dark forest dungeon in the **Formoria** campaign setting.

Note: The adventure is currently unreleased and unpublished and will be launched as part of an Anthology of Formorian advnetures to be crowdfunded in early 2026.

## Using the Living Grove

To shift the grove during play:

1. Print the Mournwood cutout map (Formoria_Mournwood_Print_Cutout_Individual_Hexes_Player.pdf).
2. Cut along the dotted hex borders.
3. Swap the positions of any two hexes.
4. Realign interior paths and connections as needed.

Alternatively, Appendix A of the adventure will include pre-built Mournwood map configurations with hexes already shifted, allowing for rapid changes without table disruption.
